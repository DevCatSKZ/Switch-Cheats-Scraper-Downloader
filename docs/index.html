What's inside

Built for the whole cheat lifecycle.

Nine-plus sources, one database

CheatSlips (scrape + official API), GBATempArchive, HamletDuFromage, Sthetix, Breeze, Chansey 60FPS/Res/GFX, MyNXCheats, ibnux and titledb — merged per build so duplicates are never lost, kept in a WAL-mode SQLite DB.

Self-updating

Every app checks GitHub for a newer build and refreshed cheats — including re-uploads without a version bump.

API-first, no reCAPTCHA

Downloads via the official X-API-TOKEN with automatic API-limit reset, and a Playwright browser fallback for the rest.

SD-card & ZIP export

Auto-detects the drive and writes the exact Atmosphère / Breeze / EdiZon layout — or packs it into a ZIP to import later.

Six languages

EN · DE · ES · FR · IT · JA in every app; the language is auto-detected and switchable.

Nothing goes missing

Builds the API reports as empty aren't silently dropped — they land under a “Not downloaded” filter. Covers, regions and versions are enriched on demand.

How it works

Collect, browse, export.

Collect

Scrape CheatSlips and import the community sources into your local SQLite database — game names, covers, regions, versions and descriptions.

Browse

Search and filter everything in the GUI. See what's downloaded, what's missing, and where each build's cheats came from.

Export

One click to the SD card or a ZIP — in exactly the layout the CFW expects. On the Switch and Android apps, the same write happens on the device.

Privacy

Your data stays yours.

Switch Cheats Scraper & Downloader collects no personal data. There are no analytics, no trackers, no ads and no user accounts on our side — the apps run entirely on your device.

Storage access

The app requests file access only to save downloaded cheat files into your emulator or SD-card folders. It never reads or uploads your other files.

Network

It connects to GitHub and the public cheat sources solely to download cheats and check for updates — nothing about you is sent.

No accounts, no tracking

Any CheatSlips credentials you enter in the Windows tool stay on your device and go only to CheatSlips to log you in. We store nothing about you.
