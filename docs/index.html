What's inside

Built for the whole cheat lifecycle.

Nine-plus sources, one database

CheatSlips (scrape + official API), GBATempArchive, HamletDuFromage, Sthetix, Breeze, Chansey 60FPS/Res/GFX, MyNXCheats, ibnux and titledb — merged per build so duplicates are never lost, kept in a WAL-mode SQLite DB.

Self-updating

Every app checks GitHub for a newer build and refreshed cheats — including re-uploads without a version bump.

API-first, no reCAPTCHA

Downloads via the official X-API-TOKEN with automatic quota reset, and a Playwright browser fallback for the rest.

SD-card & ZIP export

Auto-detects the drive and writes the exact Atmosphère / Breeze / EdiZon layout — or packs it into a ZIP to import later.

Six languages

EN · DE · ES · FR · IT · JA across every app, auto-detected and switchable — even in the Windows installer.

Nothing goes missing

Builds the API reports as empty aren't silently dropped — they land under a “Not downloaded” filter, and a re-scan that returns 0 cheats never overwrites a count you already have. Covers, regions and versions are enriched on demand from titledb, switchbrew and tinfoil.

How it works

Collect, browse, export.

Collect

Scrape CheatSlips and import the community sources into your local SQLite database — game names, covers, regions, versions and descriptions.

Browse

Search and filter everything in the GUI. See what's downloaded, what's missing, and where each build's cheats came from.

Export

One click to the SD card or a ZIP — in exactly the layout the CFW expects. On the Switch and Android apps, the same write happens on the device.
